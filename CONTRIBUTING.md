Thank you for considering contributing to ShieldAI! Your work can help make online spaces safer for women and girls in Kenya.

Quick start
1. Fork the repository and create a new branch for your change (feature/my-feature or fix/short-description)
2. Run the project locally and reproduce any issue or test your change (see README.md for setup)
3. Add tests for new functionality where appropriate
4. Submit a PR with a clear description and testing instructions

Guidelines
- Be respectful and inclusive. This project is intended to support survivors and build safer systems.
- Keep PRs small and focused â€” one logical change per PR.
- Write tests for new features and make sure existing tests pass.
- Add documentation to README or create a new docs page if you change behavior.

Issue and feature requests
- Open an issue describing the problem and include steps to reproduce.
- Tag issues with labels and provide context for why the change matters for Kenya-focused moderation.

Code of conduct
Please follow the repository's code of conduct. Be kind and respectful to others. If you'd like, I can add a formal CODE_OF_CONDUCT.md.

If you'd like me to add automation (PR template, issue templates, GitHub Actions), tell me what you'd like and I can add them next.
